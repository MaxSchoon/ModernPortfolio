# Support

Thanks for using Modern Portfolio Optimizer. Here is how to get help.

## Questions about using the optimizer

- Read the docs first. `README.md` covers installation, project structure, and
  basic usage.
- Ask a question in GitHub Discussions if Discussions are enabled:
  https://github.com/MaxSchoon/ModernPortfolio/discussions
- For general questions or collaboration proposals, email max@doc2ixbrl.com.

## Found a bug or correctness issue?

Open an issue and pick the template that fits:

- Bug report for runtime errors, data-fetch problems, cache issues, CLI
  behavior, or report generation problems.
- Optimizer correctness for numerical, financial, or constraint behavior that
  appears wrong.
- Enhancement for new features, optimizer modes, reports, or developer
  experience improvements.

Issue templates are here:
https://github.com/MaxSchoon/ModernPortfolio/issues/new/choose

## Security concerns

Do not open a public issue for a security vulnerability. See `SECURITY.md` for
private reporting through GitHub security advisories.

## What support is and is not

This is a community, best-effort, open-source project maintained on a volunteer
basis. There is no SLA. The optimizer is experimental and educational; it is
not investment advice and should not be used as a substitute for independent
financial judgment.
